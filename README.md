# 💻 Portfolio Website

A personal portfolio website designed to showcase projects, technical skills, and development journey through a clean and interactive user experience. Built with a focus on performance, responsive design, and a modern interface to create a strong digital presence.

## ✨ Key Features

Responsive design optimized for desktop, tablet, and mobile devices
Dedicated sections for projects, skills, and personal information
Interactive UI with smooth navigation and modern design elements
Project showcase with live demos and repository links
Fast-loading and performance-focused implementation

## 🧱 Tech Stack
Next.js • React • TypeScript • Tailwind CSS • Vercel
